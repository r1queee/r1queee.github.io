Project

Energy Monitor

A real-time energy monitoring device that measures consumption, stores history, and exposes data to a dashboard.

ESP32 AC Measurement Wi-Fi PCB Dashboard
GitHub Demo Back to Projects

Graphs / Dashboard

Why I built it

Goals

  • Measure household energy usage in real time (power + energy).
  • Track usage over time (daily/weekly/monthly history).
  • Expose data through a clean dashboard and API.
  • Design a compact and reliable PCB solution.

Hardware

ESP-32

  • MCU principal do projeto
  • Wi-Fi / BLE
  • Leitura ADC + cálculo RMS
  • Bluetooth para app
  • Gestão do display (I2C)
ESP32 DevKit Module

SCT-013-100

  • Sensor de corrente (CT) não-invasivo
  • Isolamento (segurança em AC)
  • Burden / escala e calibração
  • Boa opção para cargas domésticas
CT / SCT-013-100

ZMPT101B

  • Sensor de tensão AC (amostragem no ADC)
  • Ajuste de ganho/offset
  • Filtragem + estabilidade
  • Calibração com referência (carga resistiva)
Voltage / ZMPT101B

SSD1306

  • Display OLED para leitura local
  • I2C
  • Baixo consumo
  • UI simples (V / A / W)
OLED / SSD1306

APV-8-5

  • Fonte AC → DC (5V)
  • Base para regulação 3.3V
  • Estabilidade de alimentação
  • Isolamento / segurança (AC)
Power / APV-8-5

Architecture

  • Sampling + filtering to get stable RMS readings.
  • Energy integration over time (Wh / kWh).
  • Wi-Fi transport (MQTT / HTTP) to send telemetry.
  • Dashboard consumes the API and plots history.

What I focused on

  • Accuracy vs. sampling rate and noise.
  • Resilience: reconnect logic and data buffering.
  • Calibration routine and storing calibration constants.

Outcome

Results

Highlights

  • Real-time power readout and energy accumulation.
  • History stored and displayed in the dashboard.
  • Stable readings after filtering and calibration.
